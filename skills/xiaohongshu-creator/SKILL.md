# 小红书创作者

Create engaging, viral-ready Xiaohongshu content with optimized titles, structure, and formatting.

## Quick Start

1. **Determine content type** - Tutorial, review, lifestyle, travel, food, etc.
2. **Generate title** - Use title formulas for maximum click-through
3. **Structure the post** - Hook + body + CTA format
4. **Add hashtags** - Mix of popular and niche tags
5. **Optimize formatting** - Emojis, line breaks, visual flow

## Title Generation

### Viral Title Formulas

**Number + Benefit**
- "3个小技巧，让你的xxx立刻提升"
- "拒绝无效xxx！这几个方法真的太好用了"

**Question + Solution**
- "xxx总是不理想？试试这个方法"
- "为什么xxx一定要xxx？"

**Emotional Hook**
- "我不允许任何人还不知道xxx"
- "破防了！这个xxx真的太..."

**Before/After Contrast**
- "从xxx到xxx，我只做了这几件事"
- "改变后的xxx，连朋友都不认识我了"

### Title Tips

- Keep under 20 characters for mobile display
- Use emotional words: 必看, 神仙, 绝绝子, 破防了, 救命
- Add emojis at start or end for visual pop
- Create curiosity gap - don't reveal everything

## Post Structure

### Standard Format

```
【Hook】（前2句）
吸引注意力的开头，制造悬念或共鸣

【Body】（主体内容）
• 要点1
• 要点2
• 要点3

详细说明/个人体验/实操经验

【CTA】（结尾）
引导互动或分享
```

### Content Templates

**Tutorial/How-to**
1. Hook: 问题/痛点 + 承诺解决方案
2. Step-by-step with clear numbering
3. Pro tips section ("小贴士")
4. CTA: 让用户分享他们的经验

**Review/Recommendation**
1. Hook: Personal verdict upfront
2. Pros/cons breakdown
3. Who it's for / Who should skip
4. Alternatives mentioned
5. CTA: Ask for their recommendations

**Lifestyle/Story**
1. Hook: Relatable scenario or surprising fact
2. Personal journey/experience
3. Key learnings/takeaways
4. CTA: Invite similar stories

## Hashtag Strategy

### Mix Formula
- 2-3 热门大话题 (100w+ 笔记)
- 3-5 垂直细分话题 (10w-100w 笔记)
- 2-3 品牌/场景特定话题
- 1-2 响应话题 (如 #今日份小确幸)

### Placement
- Add 3-5 hashtags in the first comment for cleaner look
- Include 1-2 in the post body naturally
- End with signature hashtag block

## Emoji Usage

### Opening Emojis (Use 1-2)
💡 🔥 ✨ 🌟 👍 👏 🏰 📱 🚀

### Section Dividers
────────────────────────────
（用纯文字分隔线，更简洁专业）

### Bullet Points
• → ▶ ✓ ⚠ ⭐ 

### Closing Emojis
👋 ❤️ 💖 🎉 🙏 👌

### Best Practices
- Don't overdo - max 10-15 emojis per post
- Place at line starts for visual rhythm
- Match emoji mood to content tone
- Use → for steps, ✓ for benefits, ⚠ for warnings

## Formatting Tips

### Line Breaks
- Single break: Shift+Enter (keeps paragraph together)
- Double break: Full paragraph separation
- Use breaks every 2-3 lines for mobile readability

### Visual Structure
```
────────────────────────────
💡 核心要点
────────────────────────────

✓ 要点1：xxx
✓ 要点2：xxx
✓ 要点3：xxx

────────────────────────────
⚠ 注意事项
────────────────────────────
```

## Content Best Practices

1. **First 2 lines are critical** - Users see preview before expanding
2. **Speak like a friend** - Conversational, not corporate
3. **Add value fast** - Don't bury the lead
4. **Include personal experience** - Builds trust
5. **Ask questions** - Drives engagement/comments
6. **Respond to comments** - Boosts algorithm ranking
